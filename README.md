# HoangHa Sensitivity Optimizer

Ứng dụng Flutter phân tích thiết bị iOS và đề xuất cấu hình độ nhạy tối ưu cho Free Fire. Cung cấp chức năng sinh file `.mobileconfig` kìm hãm màn hình.

## Tính năng
- Đọc thông số thiết bị iOS thật.
- Đề xuất độ nhạy tối ưu (max 200).
- Đề xuất kích thước nút bắn giúp "hãm phanh" headshot.
- Tải file `.mobileconfig` cấu hình.
- Mở nhanh Cài đặt Trợ năng & Free Fire.

## Build IPA
Chạy Github Actions để tự động build file `.ipa` dành cho TrollStore.
